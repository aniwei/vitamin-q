# 变更：补充 QuickJS 字节码文档与 ES2020 夹具

## 为什么
项目需要权威、函数级别的 QuickJS 字节码生成文档，以及覆盖完整 ES2020 语法的夹具集合，以支持编译器开发、验证与团队上手。

## 变更内容
- 完整整理 QuickJS 字节码生成文档（全部语法、逐 opcode 语义、生成流程）。
- 为每个 ES2020 语法形式提供字节码生成路径与函数级引用。
- 输出 ES2020 全语法字节码生成的完整流程说明（从解析到发射的端到端链路）。
- 在 fixtures/es2020 下新增完整的 ES2020 夹具集合。
- 建立可重复的文档覆盖验证流程（对照源码与语法清单）。

## 影响
- 影响规格：quickjs-bytecode-docs
- 影响代码：third_party/QuickJS（只读分析）、fixtures/es2020、文档新增位置
- **BREAKING**：无
